Block AI Overview (Chrome extension)

Replace this page’s URL in the Chrome Web Store with the public URL where you host this file (for example GitHub Pages).

Summary

This extension does not sell user data. It does not send your searches or personal information to the extension developer. It works entirely on your device when you use Google Search on the sites listed in the extension’s permissions.

Data the developer collects

The developer does not operate a server for this extension and does not collect personally identifiable information, health, financial, authentication, communications, location, web history, keystrokes, or page content for analytics or resale.

Data stored on your device

The extension may store a single on or off setting (whether automatic “-AI” appending is enabled) using the browser’s storage API so your choice persists. That setting may sync through your browser’s built-in sync if you use it; it is not sent to the developer.

How the extension works

On supported Google Search pages, the extension may adjust the search URL so your query includes the “-AI” search modifier, matching what you could type yourself. This happens locally in the browser.
